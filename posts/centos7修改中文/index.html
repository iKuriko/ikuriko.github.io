CentOS7修改中文

安装中文语言包

yum install kde-l10n-Chinese

yum reinstall glibc-common

安装字体

yum groupinstall "fonts"

查看系统支持的语言（是否支持zh_CN.utf8）

locale -a | grep zh

编辑系统语言文件

vim /etc/locale.conf
LANG=zh_CN.utf8

立即生效

source /etc/locale.conf

也可以直接设置系统语言

localectl  set-locale LANG=zh_CN.utf8
